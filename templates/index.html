Simülâtör Hakkında

E-ticaret Simulator, modern e-ticaret platformlarının tüm özelliklerini barındıran kapsamlı bir simülasyon uygulamasıdır. Bu platform sayesinde:

  • Ürün yönetimi yapabilirsiniz
  • Sipariş süreçlerini deneyimleyebilirsiniz
  • Kullanıcı deneyimini test edebilirsiniz
  • E-ticaret iş akışlarını öğrenebilirsiniz

Teknik Özellikler

Platform modern web teknolojileri kullanılarak geliştirilmiştir:

  • Backend: Python Flask, SQLAlchemy
  • Frontend: Bootstrap 5, Responsive Design
  • Veritabanı: SQLite
  • Güvenlik: CSRF Protection, Secure Sessions
